# BIG FISH Payment Gateway - PHP SDK

## Version

3.10.0

## Requirements

 * PHP 5.3+

## Installation

BIG FISH Payment Gateway is available at packagist.org, so you can use composer to download this library.

```
{
    "require": {
        "bigfish/paymentgateway": "3.*"
    }
}
```
